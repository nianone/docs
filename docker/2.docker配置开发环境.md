# docker配置开发环境

docker可以很方便的部署开发环境

```bash
-d         后台启动
--name     容器名称
-m         最大内存
-p         端口映射 主机：容器
-e         设置环境变量
-v         绑定一个卷/目录
--restart always 总是随着docker的启动而启动
```

## 1.MySQL 
 
 ```text
 docker pull mysql:5.7
 docker pull mysql:8.0.19
 ```

 运行
 ```text
docker run -d \
--name mysql8.0 \
-m 1024M \
-p 3306:3306 \
-e MYSQL_ROOT_PASSWORD=MysqlPwd \
-v ~/data/mysql/data:/var/lib/mysql/ \
--restart always \
mysql:8.0.19
 ```

 ## 2.Redis

```text
docker pull redis:latest
```

```text
docker run -d \
--name redis \
-m 512M \
-p 6379:6379 \
-v ~/data/redis/data:/data \
--restart=always \
redis:latest

# 配置文件
-v ~/data/redis/redis.conf:/etc/redis/redis.conf \
```
