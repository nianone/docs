# docker配置开发环境

## 1.MySQL 
 
 ```text
 docker pull mysql:5.7
 docker pull mysql:8.0.19
 ```

 运行
 ```text
 docker run -d \
 --name mysql8.0 \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=MysqlPwd \
  -v ~/dockerdata/mysql/data:/var/lib/mysql/ \
  --restart always \
  mysql:8.0.19
 ```

 ## 2.Redis
